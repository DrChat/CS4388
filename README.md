## How to Build
Prerequisites: [Qt 5.8](https://www.qt.io/download-open-source/)

Open a terminal in the project root, and run:
```
Run once:
$ sudo apt-get install clang-3.8 libc++-dev
$ python cg-build.py setup

$ python cg-build.py build
```
